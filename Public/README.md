## Vamos a explicar un poco los controles de los juegos.

# SNAKE
Space para empezar, reiniciar o pausar el juego.
Flechitas y AWSD para movimientos.
(BUG WARNING)
Cambiar rapidamente entre dos posiciones como W y A usando previamente la D te mata. ( A las letras nos referimos a los lugares de movimiento.)
# TETRIS
Space para empezar o reiniciar.
A para llevar una flecha a la izquierda.
D para llevar una flecha a la derecha.
W para hacer que una pieza cambie su posicion.
(BUG WARNING)

# DINO
Space para en menu de inicio para empezar o al morir space para reiniciar.
Space para saltar.
(BUG WARNING)
